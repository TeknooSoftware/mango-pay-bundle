Installation & Requirements
---------------------------
To install this bundle

    composer require teknoo/mango-pay-bundle
    
And add to your AppKernel : 

    new Teknoo\MangoPayBundle\TeknooMangoPayBundle(),
    
This library requires :

    * PHP 5.5+
    * Composer
    * Symfony 2.7+
    * Mango Pay API
